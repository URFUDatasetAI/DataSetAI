Краткая инструкция

  1. Слева выбери тестового пользователя или введи его ID вручную.
  2. Под ролью заказчика создай комнату и пригласи разметчика по ID.
  3. Под ролью разметчика открой приглашенную комнату и нажми «Войти в комнату».
  4. Запроси следующую задачу и отправь JSON с результатом разметки.
