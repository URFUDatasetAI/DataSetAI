Crowdsourcing MVP

Backend + интерфейс для разметки датасетов

Этот проект дает заказчику возможность создавать комнаты для разметки, приглашать исполнителей, выдавать задачи и собирать результаты. Архитектура рассчитана на локальный запуск, корпоративное разворачивание и дальнейшее развитие API без болезненного рефакторинга.

Что делает заказчик

Создает комнаты, добавляет тестовый датасет, задает пароль, дедлайн и приглашает разметчиков.

Что делает разметчик

Видит только доступные ему комнаты, заходит в рабочую среду, берет задачи и отправляет разметку.

Что уже заложено

PostgreSQL, DRF API, mock identification, dashboard по комнате и профиль со статистикой активности.

Текущая цель MVP

Быстро дать команде рабочий контур системы, где можно руками проверить основной пользовательский путь: создать комнату, открыть ее, выполнить разметку и увидеть прогресс по работе.

Следующий естественный шаг

Подключить полноценную аутентификацию, реальную загрузку датасетов и управление жизненным циклом задач без изменения базовой структуры проекта.
