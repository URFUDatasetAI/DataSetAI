# Ideas Layer

`AGENTS/ideas/*` - это backlog-слой только для явно обсуждённых идей.

Важное правило для этого репозитория:

- не заполнять `ideas/` по собственной интерпретации документации, scenario-файлов или “кажется, это логичное направление”;
- не превращать возможные future extensions в backlog без явного продуктового обсуждения;
- добавлять запись сюда только тогда, когда идея действительно была проговорена и ей нужен статус `candidate`, `needs-discovery` или `rejected`.

Сейчас этот слой намеренно пустой по содержанию. Начинать всё равно лучше с [index.md](index.md), потому что он показывает текущее состояние buckets.
